Базовая структура информационной системы.

Что такое информационная система (ИС)?

Информационная система (ИС) — это система, предназначенная для сбора, хранения, обработки, поиска, передачи и предоставления информации в соответствии с требованиями пользователей. Это не просто компьютеры и программы; это комплекс взаимосвязанных компонентов, работающих вместе для достижения общей цели — удовлетворения информационных потребностей организации или человека.

Простыми словами, ИС превращает данныев полезную информацию и знания,на основе которых принимаются решения.

Устройство (Структура) информационной системы

Любую ИС можно представить как состоящую из нескольких взаимосвязанных компонентов. Классическая модель включает 5 ключевых элементов:

  1. Функциональные подсистемы (Процедуры и бизнес-логика)

    2. Это «мозг» системы. Они определяют что система должна делать.

    • Бизнес-правила и процедуры: Формализованные инструкции, по которым работает система (например, алгоритм расчета зарплаты, порядок согласования заказа, workflow документооборота).
    • Функциональность: Конкретные действия, которые может выполнять система (формировать отчет, создавать заявку, резервировать товар).
  2. Программное обеспечение (Программный компонент)

    3. Это «интеллект» системы, набор инструкций, который управляет аппаратной частью.

    • Системное ПО: Операционные системы, системы управления базами данных (СУБД: Oracle, MySQL, PostgreSQL), сетевое ПО.
    • Прикладное ПО: Непосредственно программы, решающие задачи пользователя (1С:Бухгалтерия, MS Office, CRM-системы, веб-браузеры).
    • Инструментальные средства: Языки программирования, среды разработки (IDE), используемые для создания ПО.
  3. Аппаратное обеспечение (Техническое обеспечение)

    4. Это «тело» системы, физические устройства, на которых работает ИС.

    • Серверы: Мощные компьютеры, хранящие данные и выполняющие основные вычисления.
    • Рабочие станции (ПК, ноутбуки): Компьютеры пользователей.
    • Сетевое оборудование: Маршрутизаторы, коммутаторы, кабели, Wi-Fi-точки доступа, обеспечивающие связь между устройствами.
    • Периферийные устройства: Принтеры, сканеры, источники бесперебойного питания (ИБП).
  4. Данные (Информационные ресурсы)

    5. Это «кровь» системы, самый ценный ресурс, который накапливается и обрабатывается.

    • Базы данных (БД): Упорядоченные наборы данных, хранящиеся на серверах.
    • Файлы: Документы, таблицы, изображения, видео.
    • Знания: Накопленный опыт, метаданные (данные о данных).
  5. Персонал (Кадровое обеспечение)

    6. Это люди, которые взаимодействуют с системой. Без них ИС бесполезна.

    • Конечные пользователи: Сотрудники, которые используют ИС для выполнения своих задач (бухгалтеры, менеджеры, кладовщики).
    • ИТ-специалисты: Администраторы, программисты, аналитики, техподдержка, которые разрабатывают, обслуживают и развивают систему.
